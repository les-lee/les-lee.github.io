# 揭秘 webpack

这里是关于 webpack 的一些文章, 详情看侧边栏目录.

webpack 是一个打包工具, 具有 all in one 的魔力, 看官方介绍可以知道,他可以把图片, svg, html, css, js... 等等等等 都打包成一个js.

很简单, 如果你没有使用过webpack 而且又被资源引用繁杂所困扰, 那么你去使用一下你就知道为什么会有这类工具诞生.
